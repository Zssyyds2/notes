git ​

在多人开发协作中git的使用必不可少，熟练掌握git操作，可以省去翻阅资料的时间，以下是总结的git中的常用操作。

git的常用操作 ​

克隆一个现有的仓库: ​

git clone <url>
1

添加所有文件到暂存区: ​

git add .
1

提交暂存区更改： ​

git commit -m "message"
1

分支的操作 ​

    //查看所有分支
    git branch
    //创建新分支
    git branch <branch-name>
    //切换到指定分支
    git checkout <branch-name>
    //创建并切换到新分支
    git checkout -b <branch-name>
    //删除分支
    git branch -d <branch-name>
